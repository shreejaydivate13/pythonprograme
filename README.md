This Python program is a simple example of taking user input and displaying formatted output. It prompts the user to enter their name and age using the `input()` function. The entered values are stored in variables called `name` and `age`. After collecting the information, the program prints a heading titled “Student Information” and then displays the user’s name and age in a clear and organized format. This program helps beginners understand how to use variables, accept input from users, and print results in Python. It is useful for learning basic programming concepts and structure.
